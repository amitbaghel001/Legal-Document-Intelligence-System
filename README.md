# 🚀 Overview

The Legal Document Intelligence System is a full-stack AI-powered platform designed to modernize legal workflows through intelligent document processing, automated legal analysis, and secure case management.

The system enables judges, lawyers, and citizens to efficiently manage legal cases while leveraging AI-driven insights such as:
- Automated legal document summarization
- IPC section extraction
- Smart case similarity analysis
- Workflow automation

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/amitbaghel001/Legal-Document-Intelligence-System.git
cd Legal-Document-Intelligence-System
```

---

## 2️⃣ Backend Setup

```bash
cd Backend
npm install
```

---

## 3️⃣ Frontend Setup

```bash
cd ../Frontend
npm install
```

---

## 4️⃣ Configure Environment Variables

Create a `.env` file inside the Backend directory:

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000
```

---

# ▶️ Running the Project

## Start Backend

```bash
cd Backend
npm start
```

For development mode:

```bash
npm run dev
```

---

## Start Frontend

```bash
cd Frontend
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
